مشروع Flask بسيط لتسجيل الدخول والتسجيل (بدون JavaScript)
=========================================

تشغيل المشروع محليًا:
1. أنشئ بيئة افتراضية (موصى به):
   python -m venv venv
   source venv/bin/activate  # on Linux/Mac
   venv\Scripts\activate   # on Windows

2. ثبت المتطلبات:
   pip install -r requirements.txt

3. شغّل التطبيق:
   python app.py

4. افتح المتصفح وزُر:
   http://127.0.0.1:5000

ملاحظات:
- غيّر قيمة app.secret_key في app.py إلى قيمة عشوائية وحفظها بشكل آمن قبل أي استخدام حقيقي.
- قاعدة البيانات بسيطة (SQLite users.db) وتُنشأ تلقائيًا داخل مجلد المشروع.
- لا توجد جافاسكربت كما طلبت.
- لو عايز تضيف صفحات أو تعديل التصميم قلّي وانا أعدلهولك.
